Indicators

  • I don't see the indicator listed on MT4
    I don't see the indicator listed on MT4

    Kindly note that since the Metatrader4 b600 update, indicators are no longer stored in C:/Program_Files/Metatrader/experts/indicators. Instead, indicators must be saved in the data folder of Metatrader4.

    Follow these steps.
    1) Open Metatrader.
    2) Click File - Open Data Folder. A browser will open.
    3) Store the .ex4 file in mql4/indicators.
    4) Restart MT4.

  • The indicator loads but displays nothing
    The indicator loads but displays nothing

    If the indicator displays nothing, make sure of the following.

    1) Make sure to enable DLL Calls in your platform
    2) Make sure to be connected to your broker account
    3) If you have renamed the indicator file, restore the original filename

  • The indicator is missing functionality
    The indicator is missing functionality

    Some indicators are recursive and call themselves. Make sure to have saved the indicator in mql4/indicators, and not in a sub-folder. If the indicator is in a sub-folder, the recursive call will fail and some features of the indicator will be missing.

  • The indicator crashes my metatrader platform!
    The indicator crashes my metatrader platform!

    Some indicators, for example pz-wolfe-waves or pz-harmonic-trading, depend on the zig-zag indicator. Make sure to have the zig-zag indicator installed and compiled in mql4/indicators. You can download the zig-zag indicator from here

  • I get a "DLL not allowed" error!
    I get a "DLL not allowed" error!

    Make sure to enable DLL calls in your metatrader plaform.

  • I don't see any backtest results for indicators!
    I don't see any backtest results for indicators!

    Indicators are software plug-ins which only load on your chart and plot visual information, but indicators can't trade for you and thus cannot be backtested in the strategy tester.

  • Which are the best parameters?
    Which are the best parameters?

    When it comes to indicators, optimization is not really necessary. The best parameters are the default parameters: that is why they were selected as such by the developer.

  • I get a license denied error!
    I get a license denied error!

    The indicator filename has your license number appended to it. If you renamed the indicator file the license check will fail. If so, please restore the original filename of the indicator. If the problem persists, you might be behind a firewall and you should look into that.

  • Does it work on renko charts?
    Does it work on renko charts?

    All indicators will load into an offline chart, but there is no point. Renko charts are not price action, but an over-simplification of reality: the market is bidimensional, price and time are just as important. If you use renko charts, our indicators will not be useful to you.

  • Does it work on offline mode?
    Does it work on offline mode?

    No, it won't. All of our products require you to be connected to your broker account. If you are not connected to your account, the indicator will not load.

  • Does it implement alerts?
    Does it implement alerts?

    Yes! All indicators implement visual, sound, email and push alerts.

  • How do I configure mail alerts?
    How do I configure mail alerts?

    Metatrader uses your own mail server to send email alerts. You must enter your mail server and credentials into your metatrader settings. Then, enable mail alerts in the indicator inputs.
